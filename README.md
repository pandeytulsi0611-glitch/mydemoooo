Project Files
1️.gadget_launch.html
A webpage designed to display the launch of a tech drone with full specifications.
Features


Proper use of HTML5 structure (<!DOCTYPE html>, <head>, <body>).


Heading and descriptive paragraph with HTML formatting tags:


<strong>, <em>, <ins>, <s>, <br>, <hr>




Images included using absolute and relative paths.


Ordered (<ol>) and unordered (<ul>) lists.


Table displaying drone models and features.


Subscript and superscript usage.


<pre> tag for preformatted text.



2️. layout.html
A layout-based webpage demonstrating semantic HTML tags and form elements.
Includes


Semantic tags:
<header>, <main>, <section>, <article>, <aside>, <footer>


A sample link opening in a new tab.


A structured HTML form containing:


Text inputs


Checkboxes


Select dropdown


Textarea




Embedded YouTube video using an <iframe>.
